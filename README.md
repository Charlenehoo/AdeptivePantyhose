# 命名规范文档 (Naming Conventions)

## 概述

本文档定义了 C++项目中应遵循的命名规范，确保代码风格统一、可读性强、易于维护。

---

## 一、类型命名 (Type Names)

### 1. 类名 (Class Names)

- **规范**: PascalCase
- **示例**:
  class PlayerCharacter;
  class InventorySystem;

### 2. 结构体名 (Struct Names)

- **规范**: PascalCase
- **示例**:
  struct GameSettings;
  struct VertexData;

### 3. 枚举类型 (Enum Types)

- **规范**: PascalCase
- **示例**:
  enum class WeaponType;
  enum class LoadState;
